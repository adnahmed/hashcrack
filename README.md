# crackq.me

## Description

crackq.me is built with hashcat and Next.js.

## Requirements

- Postgres
- Node
- Next.js

## Getting Started

First, install dependencies:

```bash
npm install
```

then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
